Selecting a PDF

  1. Drag and drop a PDF file to PDF Split, or click browse to select the desired file.
  2. Enter the page number or page ranges that you would like to split. For example: 1, 2-3, 4-6, 7-9, 10.
    This will create 5 documents:
    Document 1 contains page 1 from the original document.
    Document 2 contains pages 2 and 3 from the original document.
    Document 3 contains pages 4, 5, and 6 from the original document.
    Document 4 contains pages 7, 8, and 9 from the original document.
    Document 5 contains page 10 of the oirginal document.
  3. Click "Split"

PDF Split automatically saves the new files in the same folder/directory where the original document is saved.
